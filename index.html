Welcome

Welcome to my portfolio! This project showcases an innovative vending machine system linked to a website. It’s designed to reward productivity with snacks, integrating advanced features like real-time updates and Google authentication.

Project Overview

This project motivates students and promotes productivity through a connected vending machine and website. Key features include:

  • Google Sign-In for secure user authentication.
  • Snack cart with dynamic selection and real-time updates.
  • Secure point-based purchase system.
